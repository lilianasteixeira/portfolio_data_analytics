# Billboard Hits Songs Data Analytics Portfolio

## Overview
This project analyzes Billboard hits songs using a dataset sourced from Kaggle. The goal is to explore trends, popularity, genres, and other metrics related to iconic songs that have topped the charts. Through data visualization and analytics, this portfolio demonstrates skills in data handling, business intelligence tools, and dashboard creation.

## Dataset
The dataset used is the [Billboard Hits Songs Dataset](https://www.kaggle.com/datasets/dem0nking/billboard-hits-songs-dataset) from Kaggle, which includes information on popular songs such as:
- Title
- Artist
- Genre
- Release Date
- Duration
- Album
- Popularity Score

The raw data is stored in `Iconic_Songs_Dataset.csv`.

## Tools and Technologies
- **PowerBI**: For creating interactive reports and visualizations.
- **CSV Dataset**: Primary data source.

## Setup and Installation
### Prerequisites
- PowerBI Desktop (for viewing/editing the .pbix file)


### Viewing PowerBI Report
1. Open `PowerBI/billboard_songs.pbix` in PowerBI Desktop.
2. Alternatively, view the PDF export at `PowerBI/billboard_songs.pdf`.

## Usage
- Interact with the PowerBI report to analyze song metrics, trends, and insights.
- Use the visualizations to identify top artists and popular genres.

## Insights and Visualizations
This project includes dashboards that highlight:
- Genre distribution
- Top-performing artists and genres
- Duration vs Popularity
